#### 1. Objeto del Contrato

#### 2. Alcance del Software

El software "InventarioCajaApp" incluye las siguientes funcionalidades:

- **Gestión de Inventarios:** Permite registrar y gestionar productos con detalles como categoría, marca, cantidad, proveedor, costo, fechas de registro y vencimiento, y precios de venta.
- **Registro de Ventas:** Registra las ventas realizadas, incluyendo detalles como producto, cantidad, método de pago, cliente, y fecha.
- **Gestión de Proveedores y Clientes:** Mantiene un registro de proveedores y clientes con sus respectivos datos de contacto.
- **Escaneo de Códigos:** Permite escanear códigos de barras y QR para agregar productos al inventario o realizar ventas.
- **Generación de Informes:** Ofrece informes y dashboards sobre ventas, inventarios, y otros aspectos operativos.
- **Interfaz Multiplataforma:** Desarrollada con Flet, permite el uso del software en múltiples dispositivos y sistemas operativos.
